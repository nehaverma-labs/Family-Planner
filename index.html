Connect Google Sheet

All edits sync to a Sheet you own. Share the same HTML file + this URL with any family member and they see live data.

  1. Create a new Google Sheet and open Extensions → Apps Script.
  2. Paste in the Code.gs script provided.
  3. Click Deploy → New deployment → Web app. Set "Execute as: Me" and "Who has access: Anyone".
  4. Authorize the script (click Advanced → Allow — this is normal for your own scripts).
  5. Copy the URL ending in /exec and paste below.
